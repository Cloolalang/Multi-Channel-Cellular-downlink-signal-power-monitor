Controls

Master switch and shortcuts match the Sector 1 Controls group from flows. Default PT_SERIAL_PORT=COM40, PT_MOCK_MODEM=false: real modem RX parses +QRXFTM (2nd value = RSSI dBm). Use PT_MOCK_MODEM=true for development without hardware. AT+QRFTESTMODE runs once at startup, then one AT+QRXFTM per enabled channel per pass, round-robin. PT_MODEM_QRXFTM_SCAN=false stops auto scans. PT_SCAN_CHANNEL_DELAY_SEC (default 1s) spaces each channel so the modem can answer before the next TX. PT_SCAN_ROUND_DELAY_SEC pauses after each full pass.
